### Gateway Server

**Build**:
```mvn package```

**Usage**:
``` java -jar <jarname> <TCP server port> <UDP server port> <UDP multicast port> <UDP multicast group address>```

### TCP client

**Build**:
```mvn package```

**Usage**:
```java -jar <jarname> <TCP server> <TCP server port>```

### UDP client

**Build**:
```mvn package```

**Usage**:
```java -jar <jarname<Multicast Group address<Multicast port>```
